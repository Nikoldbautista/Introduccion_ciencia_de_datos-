# 💳 Análisis de Riesgo Crediticio y Gestión de Clientes

Este repositorio contiene el material de clase y los conjuntos de datos utilizados para el estudio del comportamiento financiero, el procesamiento de carteras de clientes y la detección de patrones de impago.

---

## 🎯 Objetivos del Proyecto

El desarrollo de este repositorio busca cumplir con las siguientes metas estratégicas:

* **Identificación de Riesgo:** Analizar variables críticas que influyen en el cumplimiento de pagos para predecir posibles escenarios de *default*.
* **Perfilamiento Demográfico:** Segmentar la base de clientes para entender cómo factores como educación, edad y género impactan en el comportamiento crediticio.
* **Optimización de Datos:** Implementar flujos de limpieza y transformación que conviertan datos crudos (CSV/Excel) en información lista para la toma de decisiones.
* **Generación de Insights:** Producir reportes visuales y estadísticos que faciliten la comprensión de tendencias financieras complejas.

---

## 📚 Contenidos del Curso

A lo largo del desarrollo de estos archivos, se cubren los siguientes temas:

1.  **Limpieza de Datos:** Manejo de valores nulos, duplicados y formatos de archivos.
2.  **Análisis Exploratorio (EDA):** Identificación de correlaciones y patrones de comportamiento.
3.  **Visualización Estadística:** Creación de gráficos avanzados para la distribución de créditos.
4.  **Modelado de Riesgo:** Preparación y validación de datos para la predicción de incumplimientos.
